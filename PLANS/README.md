# PLANS

`PLANS/` contains human-authored planning material for this repository.

## Rules

- Treat files in this directory as authoritative for repository-local scope and
  sequencing unless a maintainer says otherwise.
- Do not use this directory for agent scratch work.
- Do not rewrite an existing plan by default.
- If a task requires analysis before implementation, keep that material in
  `satellite/.notes/` unless explicitly instructed to edit a plan here.

## Contents

- `PLAN_*.md`
  repository work plans
- `prompts/`
  plan support material, not runtime policy
- `testing/`
  notes on policy binding for this repository

## Boundary

`PLANS/` describes work on `agent-backbone` itself.
It is not the plan location for consuming repositories.
