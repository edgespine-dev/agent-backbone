# Satellite Notes

`satellite/.notes/` is reserved for agent working material in this repository.

## Use this directory for

- implementation breakdowns
- test notes
- risks and assumptions
- open questions
- analysis that should not modify authoritative plans or production repository
  surfaces

## Do not use this directory for

- final policy text
- canonical skill content
- registry data
- install scripts
- human-authoritative plans

## Relation to other surfaces

- `PLANS/` is human-authoritative.
- `satellite/.notes/` is agent-owned and disposable.
- `skills/`, `registry/`, `policies/`, and `scripts/` are production repository
  surfaces.
